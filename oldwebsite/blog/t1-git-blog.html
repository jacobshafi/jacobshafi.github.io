Version Control is especially useful when collaborating with a number of people on one specific task. Version control contributes to clarity in code and diminshes any chance of overwrite by collaborating piers. This ingenious invention has made open source coding possible for everyone around the world who wishes to contribute code taylored to their needs.

Git is a device that derives from this concept of version control. Git stores all changes you have made to a file in a log. Every change you make in a file it is stored on the local computer and can be accessed any time.

GitHub is the most useful medium to store code because it contributes to a collaborative experience that software engineering inherently embodies. This medium makes tracking changes on the internet and boundless contributions to code possible. It is by far the most populat hub in which coders share and tweak code finely tuned to their needs.
